Math Object

The Math object in Javascript allows us to perform mathematical functions on data without having to do them manually. This can be incredibly helpful when needing to display fun charts and graphs that display info about ecommerce sales, students' progress reports, and more.

The Math.random() function returns a floating-point, pseudo-random number in the range [0, 1] that is, from 0 (inclusive) up to but not including 1 (exclusive), which you can then scale to your desired range.
